# Shot-lista: Hapkido Sverige Promotionvideo

**Produktion:** WKF/Hanminjok Hapkido Sverige
**Format:** 60s huvudvideo (16:9) + 20-30s kort version (9:16)
**Inspelningsplats:** Dojang (träningslokal)
**Utrustning:** Mobilkamera med slowmotion-kapacitet (120fps/240fps)
**Musik:** Lugn men positiv, varierat tempo

## Detaljerad shot-lista per scen

---

### INTRO (0-5 sekunder) -- Hook

**Syfte:** Fånga tittarens uppmärksamhet direkt. Kraftfull, visuellt slående teknik.

---

#### S01 -- Slowmotion hook-teknik

| Fält | Detalj |
|------|--------|
| **Scen/Akt** | INTRO (0-3s) |
| **Beskrivning** | En utövare utför en kraftfull hög spark (dollyo chagi eller bandal chagi) ELLER ett dynamiskt kast mot partner. Partnern faller kontrollerat. Hela rörelsen ska synas från ansats till avslut. Dobok ska röra sig i luften. |
| **Kameravinkel** | Helbild, låg vinkel (kamera ca 30 cm från golvet, riktad snett uppåt). Utövaren ska se kraftfull och dominant ut. |
| **Kamerarörelse** | Statisk. Låt tekniken göra jobbet. |
| **Hastighet** | 240fps slowmotion (spelas upp i ca 10x slow = 3s slutresultat från 0.3s realtid) |
| **Format** | Båda -- filma centrerat med luft ovanför och på sidorna, så det fungerar i både 16:9 och 9:16 beskärning |
| **Utövare** | 1 erfaren vuxen utövare (svart bälte eller mörkt bälte) + 1 fallpartner vid kast. Välj den med bäst teknik och mest visuellt tilltalande rörelse. |
| **Plats** | Dojang, mitten av mattan. Bakgrund ska vara ren -- inga väskor, flaskor eller stökiga hörn synliga. |
| **Klipplängd** | 3 sekunder |
| **Prioritet** | MASTE |
| **Anmärkning** | Filma minst 5-8 tagningar. Testa med och utan kiai (kamprop). Partnern MASTE veta hur man faller säkert. Kontrollera ljus -- undvik hård skugga i ansiktet. Om lokalen har fönster på en sida, placera utövaren så ljuset kommer framifrån/snett. Denna shot DEFINIERAR videons kvalitet. |

---

#### S02 -- Slowmotion reaktion/kiai

| Fält | Detalj |
|------|--------|
| **Scen/Akt** | INTRO (3-5s) |
| **Beskrivning** | Närbild på utövarens ansikte precis vid eller strax efter teknikens avslut. Fokus på uttrycket -- koncentration, kraft, beslutsamhet. Eventuellt kiai (kamprop) fångas med öppen mun och intensiv blick. |
| **Kameravinkel** | Närbild (ansikte och axlar), rakt framifrån eller 45 grader. |
| **Kamerarörelse** | Statisk |
| **Hastighet** | 120fps slowmotion |
| **Format** | Båda -- centrerad komposition |
| **Utövare** | Samma utövare som S01 |
| **Plats** | Dojang, samma setup som S01 |
| **Klipplängd** | 2 sekunder |
| **Prioritet** | MASTE |
| **Anmärkning** | Filma detta direkt efter S01 med samma utövare. Kan vara en separat tagning där kameran zoomar in (eller en andra mobilkamera om tillgänglig). Se till att bakgrunden inte är distraherande. Suddig bakgrund (porträttläge) kan vara ett alternativ. |

### AKT 1 (5-20 sekunder) -- Ursprung och filosofi

**Syfte:** Lugna, vackra bilder som visar respekt, tradition och Hapkidos rötter. Tempot sänks efter hook:en.

---

#### S03 -- Gemensam bukning (Charyeot - Kyeongnye)

| Fält | Detalj |
|------|--------|
| **Scen/Akt** | AKT 1 (5-8s) |
| **Beskrivning** | Hela gruppen står i linje, hör kommandot "Charyeot" (givakt), sedan "Kyeongnye" (buga). Alla bugar samtidigt, synkroniserat. Kameran fångar hela raden framifrån eller snett från sidan. |
| **Kameravinkel** | Vidvinkel, rakt framifrån (kamera i mitten) eller 30 grader från sidan för djup i bilden. Kamera i midjehöjd. |
| **Kamerarörelse** | Statisk ELLER långsam panorering från vänster till höger längs raden. |
| **Hastighet** | Normal (24/30fps) |
| **Format** | 16:9 (bred bild behövs för hela raden) |
| **Utövare** | Hela gruppen, minst 8-10 personer. Blandade åldrar och bälten. Instruktör framför gruppen. |
| **Plats** | Dojang, längs mattans långsida. Flaggor/banderoller i bakgrunden om möjligt. |
| **Klipplängd** | 3 sekunder |
| **Prioritet** | MASTE |
| **Anmärkning** | Timing är allt. Öva bukningen 2-3 gånger innan filmning så alla bugar i exakt samma takt. Svarta dobokar ska vara rena och ordentligt knutna. Kontrollera att ingen tittar åt fel håll. |

| **Kameravinkel** | Närbild, rakt framifrån eller svagt ovanifrån (45 grader nedåt). Bilden visar midja, händer och bälte -- inget ansikte. |

| **Hastighet** | Normal, eventuellt lätt slowmo (60fps) för att ge en mjuk känsla |
| **Format** | Båda -- centrerad komposition fungerar i bägge format |
| **Utövare** | 1 utövare (valfri ålder). Överväg att filma med olika bältesfärger och välja i klipp. |
| **Plats** | Dojang, nära vägg eller med ren bakgrund. |
| **Klipplängd** | 2 sekunder |
| **Prioritet** | MASTE |
| **Anmärkning** | Filma 3-4 versioner: ett vitt bälte (nybörjare, symbolik), ett svart bälte (mästerskap), eventuellt ett barns bälte. Händerna ska vara rena. Bältet ska vara nytt/rent om möjligt. |

### AKT 2 (20-40 sekunder) -- Träningen: energi, mångfald, gemenskap

**Syfte:** Visa Hapkidos bredd -- att alla kan delta. Energin ökar. Fokus på glädje, mångfald i åldrar och gemenskap.

| Fält | Detalj |
|------|--------|

### AKT 3 (40-50 sekunder) -- Teknisk bredd

**Syfte:** Visa Hapkidos unika tekniska bredd. Snabbare klipp. Varje teknikområde representerat.

#### S17 -- Slag-kombination

| Fält | Detalj |
|------|--------|
| **Scen/Akt** | AKT 3 (40-42s) |
| **Beskrivning** | En utövare utför en snabb slagkombination: t.ex. jab-cross-hook eller liknande Hapkido-slag mot mitts/kudde eller i luft. Kraftfull och explosiv. |
| **Kameravinkel** | Halvbild, 45 grader. Händerna och överkroppen ska synas tydligt. |
| **Kamerarörelse** | Statisk |
| **Hastighet** | 120fps slowmotion |
| **Format** | 16:9 |
| **Utövare** | 1 erfaren utövare + eventuellt 1 person som håller mitts. |
| **Plats** | Dojang |
| **Klipplängd** | 2 sekunder |

## Praktiska tips for inspelningsdagen

### Före inspelningen
- [ ] Ladda mobilen till 100% (slowmotion drar batteri)
- [ ] Töm lagringsutrymme (slowmotion-klipp är STORA -- räkna med minst 10-15 GB)
- [ ] Ta med extra laddare/powerbank
- [ ] Ta med tejp (för att markera kamerapositioner på golv)
- [ ] Ta med mikrofiberduk (rengör linsen ofta)
- [ ] Skriv ut denna shot-lista och ta med en penna för avbockning
- [ ] Samla in samtyckesbanketter från barns vårdnadshavare
- [ ] Kontakta utövare i förväg: "kom i ren dobok"

### Kamerainställningar
- **Huvudvideo:** 4K 30fps (eller 1080p 60fps som alternativ)
- **Slowmotion:** 1080p 120fps ELLER 1080p 240fps
- **HDR:** AV (kan orsaka problem i redigering)
- **Stabilisering:** PA om tillgänglig
- **Exponering:** Lås exponeringen (tryck och håll på skärmen) speciellt vid slowmotion
- **Rutnät:** PA (hjälper med komposition)

### Under inspelningen
- Rengör linsen mellan varje shot-grupp
- Kontrollera resultat regelbundet -- bättre att filma om direkt
- Ge tydliga instruktioner: "Vi filmar tre tagningar, sedan går vi vidare"
- Ha en person som "produktionsassistent" som håller koll på shot-listan
- Håll tempot uppe -- utövarna tappar energi om det tar för lång tid
- Filma alltid 5 sekunder EXTRA i början och slutet av varje shot (ger utrymme i klipp)

### Ljus
- Alla lampor i lokalen PA
- Undvik blandljus (dagsljus + lysrör ger grönstick)
- Om möjligt, filma med dagsljus framifrån (fönster bakom kameran)
- Slowmotion kräver MINST dubbelt så mycket ljus som vanlig video

### Ljud
- Även om musik läggs på i redigering: spela in RENT ljud
- Kiai (kamprop) och slag mot mitts kan användas som ljudeffekter
- Be folk vara tysta under slowmotion-tagningar (även om ljudet inte används)
